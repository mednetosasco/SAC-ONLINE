Passo a Passo Escrito – Plataforma Web MedNet

Bem-vindo(a) ao guia completo e fácil de seguir para usar a Plataforma Web MedNet.

Cadastrar um Novo Colaborador

Adicione novos colaboradores à plataforma com facilidade seguindo este guia passo a passo:

  • Entre na Plataforma Web MedNet e clique em Novo Funcionário para começar.
  • Preencha todos os campos obrigatórios (marcados com *) com cuidado e atenção aos detalhes.
  • Clique em Salvar para finalizar o cadastro com sucesso.
  • Dica Útil: Se o colaborador já trabalha na empresa, adicione a matrícula eSocial, o código da categoria e a data exata de admissão. Para novos colaboradores, clique em Continuar para prosseguir.

Estagiário, Sócio-Proprietário ou PJ

Esses tipos de cadastro não são enviados ao eSocial. Configure-os da seguinte maneira:

  • No campo Matrícula eSocial, insira PJ ou Estagiário, dependendo da situação.
  • No campo Código da Categoria eSocial, escolha o código correto:
    • Sócio-Proprietário: 701, 723, 741, etc.
    • Estagiário: 901.

Atualizar Cadastro de Funcionários

Mantenha os dados dos funcionários sempre atualizados e precisos. Atenção: Problemas comuns incluem a falta de matrícula eSocial, código da categoria ou data de admissão.

  • Acesse a Plataforma Web e navegue até Funcionário > Cadastro de Funcionários.
  • Clique no nome do funcionário que precisa de atualização.
  • Adicione ou corrija os dados ausentes: matrícula eSocial, código da categoria e data de admissão.
  • Confira se todos os funcionários ativos estão cadastrados. Se não, cadastre-os usando o passo anterior.

Gerar um Atendimento

Crie atendimentos de forma rápida e bem organizada com estes passos:

  • Acesse a Plataforma Web e clique em Abertura de Atendimento.
  • Selecione o nome do colaborador na lista disponível.
  • Escolha as opções certas:
    • Tipo de atendimento desejado.
    • Contrato correspondente ao colaborador.
    • Local onde o atendimento será realizado.
  • Clique em Gerar Atendimento para confirmar.
  • Baixe a guia gerada e revise os dados. Nota: Não é necessário imprimir a guia, apenas verifique!

Mudança de Função

Atualize a função de um colaborador de maneira simples e eficiente:

  • Acesse a Plataforma Web e vá para Funcionário > Cadastro de Funcionários.
  • Clique no nome do funcionário a ser atualizado.
  • No quadro Setor/Função, selecione a nova função e clique em Salvar.
  • Atenção: Se houver mudança de riscos, abra um kit e envie o funcionário para o Exame de Mudança de Risco. Se não houver, o exame não é necessário.

Desligamento de Funcionário (Sem Exame Demissional)

Desligue um funcionário sem exame demissional seguindo este processo:

  • Acesse a Plataforma Web e vá para Funcionário > Cadastro de Funcionários.
  • Clique no nome do funcionário a ser desligado.
  • Navegue até a aba Status e clique em Inativar para concluir.

Extrair Cronograma de Exames Vencidos ou a Vencer

Gerencie os exames dos colaboradores com facilidade:

  • Acesse a Plataforma Web e vá para Funcionário > Cronograma de Exames.
  • Clique em Exportar Excel para baixar o arquivo com os dados.
  • Revise as datas de exames vencidos e a vencer com atenção.
  • Certifique-se de que todos os funcionários ativos estão cadastrados. Se não, cadastre-os.

Extrair Ordem de Serviço por Função

Baixe a Ordem de Serviço de forma prática e rápida:

  • Acesse a Plataforma Web e vá para Funcionário > Cadastro de Funcionários.
  • Clique no nome do funcionário desejado.
  • Navegue até a aba Ordem de Serviço.
  • Selecione o Contrato/Unidade correspondente.
  • Clique em Download para obter o arquivo em PDF.

Baixar ASO

Acesse os ASOs dos colaboradores de maneira fácil:

  • Acesse a Plataforma Web e vá para Funcionário > Visualização de Atendimentos.
  • Se o ASO desejado não estiver disponível, siga estes passos adicionais:
    • Vá para Cadastro de Funcionários.
    • Clique no nome do colaborador.
    • Navegue até a aba Atendimento para localizar o ASO.

Exame de Retorno ao Trabalho

É feito sempre que o colaborador ficar por 30 dias ou mais afastado por motivo de saúde.

Importante:

  • Oriente seu funcionário a comparecer para o exame de retorno ao trabalho no 1º dia útil após a cessação do benefício previdenciário, munido do último "COMUNICADO DE DECISÃO DO INSS" e de eventuais laudos/exames do médico assistente do colaborador;
  • Durante o afastamento a empresa deve se comunicar com o colaborador e buscar saber se ele está se recuperando bem. Se não estiver recuperado, recomende-o a entrar no app "Meu INSS", ligar no 135 ou ir pessoalmente à agência do INSS e fazer o Pedido de Prorrogacão de Beneficio 15 DIAS ANTES de acabar o beneficio. O INSS sempre prorroga e isso evita transtornos para a empresa e falta de pagamento de beneficio ao funcionário.
